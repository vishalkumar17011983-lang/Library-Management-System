# Library Management System

A simple Library Management System built using Python and MySQL. This project allows users to manage books in a library through a menu-driven interface.

## Features

* Add a new book
* View all books
* Search books by title
* Borrow books
* Return books
* Delete books
* View total number of books
* View available books
* View borrowed books
* Check library status

## Technologies Used

* Python
* MySQL
* mysql-connector-python

## Installation

1. Install Python.
2. Install MySQL Server.
3. Install the required package:

```bash
pip install mysql-connector-python
```

4. Update the MySQL username and password in the code.
5. Run the program:

```bash
python library.py
```

## Database

The program automatically creates:

* Database: `library_db`
* Table: `library`
